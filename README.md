# Nigerian Sign Language (NSL) Recognition AI

## Overview
This project builds an AI system to recognize and translate Nigerian Sign Language (NSL) from video into text.

## Problem
Deaf individuals often lack access to accessible digital tools. Existing solutions like captions are insufficient for many users.

## Solution
This system uses computer vision and machine learning to interpret sign language gestures and convert them into text.

## Features
- Real-time video recognition
- Pose estimation using MediaPipe
- Sequence modeling using deep learning
- Scalable for future translation systems
